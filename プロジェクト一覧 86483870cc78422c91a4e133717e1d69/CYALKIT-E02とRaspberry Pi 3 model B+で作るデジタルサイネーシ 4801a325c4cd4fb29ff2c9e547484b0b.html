CYALKIT-E02とRaspberry Pi 3 model B+で作るデジタルサイネージ

導入

　研究室でCYALKIT-E02を貸してもらう機会があったので使ってみました．このデバイスはCYBLE-022001-00と温湿度センサを太陽光パネルで駆動し，取得したセンサ値をBluetoothのビーコン機能で送信するものになっています．大きさは直径20 mmほど．小さい．詳しくは以下のサイトへ．

とりあえず動かしてみる

　何はともあれ動かしてみましょう．動かした環境はブラインドをしめ切り蛍光灯だけを付けた状態です．CYALKIT-E02にはインジケータなどがないので動作しているかどうかは受信側で確認する必要があります．そこで手持ちのiPadにCypress BLE-Beaconをインストールしてセンサ値を受信しているか確認することにしました．ビーコン機能を使用しているためアプリ側での設定は特に必要ないです．

　見事に動作していますね．部屋の温度が25.9 ℃，湿度が約75 %であることが確認できます．受信頻度はおおよそ1分間隔．研究室部屋のエアコンの設定温度が25度なのでまともに動作していそう．

送信データをラズパイで受信しデジタルサイネージを作る

　ちゃんと動作していることが確認できたので，送信データをラズパイで受信しinfluxDBに保存．保存後データベースの時系列データをGrafanaでグラフ化しデジタルサイネージを作ることにしました．

準備物

　先程のCYALKIT-E02．モニタの下（おそらく誰も手を触れない場所）に両面テープで動かないように設置．

　

受信ラズパイは家に転がっていたRaspberry Pi 3 model B+を使用．後述しますがbeacontoolsが最新版だと動作しなかったので注意．

受信側コードの作成

　受信側のコードはPythonで書いていきます．こちらのサイトをもとにラズパイで受信したコードをInfluxDBに投げるコードを記述．参考サイトが詳しいのでここでは触れません．

　CYALKIT-E02からのデータを受信するため，CYALKIT-E02がどのようにデータを送信しているのかを確認．ここではiBeaconフォーマットを選択しました．データシートp.66が詳しいです．データシートによると，温湿度の情報は送信パケットのうちminor部の16 bitsを温度8 bits，湿度8 bitsに分けて送信しているらしいです．したがって，受信側で受け取ったminor部の情報を右に8 bitsビットシフトし0xFFと論理積を取ったものを湿度，そのまま受け取け取り0xFFと論理積を取ったものを温度としました．

def callback(bt_addr, rssi, packet, additional_info):
    humidity=(additional_info['minor']>>8)&0xFF
    temp=(additional_info['minor'])&0xFF

また，受け取った温湿度値は補正をかける必要があるらしく，データシートp.67に記載されている式で補正を行いました．

data['Humidity']=int(125*(humidity*256)/65536-6)
data['Temp']=int(175.72*(temp*256)/65536-46.85)

ちなみに，詳しく書きませんがmajor値には他のCYALKIT-E02と識別するための識別番号が2 bitsで入っています．

　次にbleビーコンを受信するプログラムを追加．受信には以下のbeacontoolsを使用しました．

しかしこのbeacontools，最新版が2.1.0なのだがラズパイ3b+だとうまく動作してくれません．特にエラーコードが出るわけでもないのですがbeacontoolsのcallback関数が動作してない様子です．困っているとこんなissueを発見．

どうやら過去にラズパイ3b+で動作しなかったためパッチが当てられたらしいです．気を取り直して過去バージョンをインストール．

このコードをもとにプログラムを作成すると受信できるようになりました🎉

環境は以下のとおり．

  • Python…2.7.16
  • beaconntool…1.3.1

あとはこちらのサイトをもとにGrafanaの設定をするだけです．

完成したデジタルサイネージ

　実際に作成したデジタルサイネージです．グラフの端から端までが3時間になっています．上が温度変化．下が湿度変化．こうやってみると湿度が周期的に変化しているのがわかります．温度変化にほとんど差がないことから空調設備がサボらずに働いていることがわかります．

研究室の温湿度変化を可視化することで環境づくりの助けになればいいですね．

おまけ

　使用している最中にラズパイ3b+とCYALKIT-E02の通信が途切れることがありました．なのでラズパイ3b+でcronを使用して5分おきに受信側コードを実行し再接続するように設定しました．また開いたPythonプロセスを終了するため，毎日7時にPythonプロセスをkillするように設定しました．

- /5 * * * * sudo python3 /home/usr/ドキュメント/blescanner_CYALKIT_E02.py
00 07 * * * pkill -f pytho[n] && /path/to/startscr
